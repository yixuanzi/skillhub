# SkillHub

AI Agent Skills Marketplace & Management Platform.

## Overview

SkillHub is a platform for discovering, sharing, and managing AI agent skills.

## Features

- 🔍 Search and discover skills
- 📦 Install skills with one command
- 🚀 Publish your own skills
- 🔄 Update and version management

## Quick Start

```bash
# Install a skill
clawhub install my-skill

# Search for skills
clawhub search "keyword"

# Publish a skill
clawhub publish ./my-skill
```
